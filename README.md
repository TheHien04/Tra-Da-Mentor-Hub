# 🎯 Trà Đá Mentor

> A comprehensive mentorship and skill development management platform

## 📖 About

**Trà Đá Mentor** is a modern web-based platform designed to support mentorship programs 
and skill development initiatives. The application provides an intuitive interface that 
enables complete management of the mentoring process, from onboarding mentors and mentees 
to analyzing learning performance.

## ✨ Key Features

✅ **Mentor & Mentee Management**
- Add, update, and delete mentor and mentee profiles
- Track expertise, experience, and maximum mentee capacity
- Manage mentee learning progress and goals

✅ **Group Management**
- Create mentoring groups with assigned mentors
- Track group meetings, sessions, and progress
- Monitor group performance metrics

✅ **Schedule & Session Management**
- Manage upcoming mentoring sessions
- Track meeting schedules and availability
- Organize group activities and workshops

✅ **Analytics & Reporting**
- Comprehensive dashboard with key metrics
- Performance charts and completion rates
- Detailed reports by specialty and track
- Track mentee progress over time

✅ **Testimonials & Feedback**
- Collect and manage mentee testimonials
- Categorize feedback by track (career, personal, soft skills)
- View success stories and impact metrics

✅ **Modern User Interface**
- Beautiful, professional design
- Professional color system and animations
- Fully responsive and mobile-optimized

## 🛠️ Tech Stack

**Frontend:**
- React 19.1.0
- TypeScript 5.8.3
- Vite 6.3.5
- React Router 7.6.0
- Axios 1.9.0
- React Icons 5.5.0
- Recharts 2.15.3 (charts & graphs)
- React Toastify 11.0.5 (notifications)

**Backend:**
- Node.js + Express
- MongoDB (recommended)
- RESTful API

**Styling:**
- CSS3 with modular design system
- Forest Green (#0a4b39) as primary color
- Smooth animations and transitions
